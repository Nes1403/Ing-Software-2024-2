Para correr el Script1 debemos ejecutar el comando pyuthon3 Script1.py en el directoria en donde se encuentre el escript
Despues nos van a pedir el nombre del jugador 1 y el nombbre del jugador 2

Despues se va a legeir a un jugador al azar para que saque y se va a imprimir el marcador con el siguiente formato

¡jugador 1 sacará primero!

Jugador 1 (nombreDelJugador1) 	 Sets: 0 	 Juegos: 0 	 Puntos: 0
Jugador 2 (nombreDelJugador2) 	 Sets: 0 	 Juegos: 0 	 Puntos: 0

Entonces el programa preguntara quien de los dos jugadores anoto un punto y debemos escribir el nombre del jugador que hizo el punto. En caso de haber escrito mal un nombre o si escribimos cualquier otra cosa el programa arroja un error y con ayuda de la estructura try-except pide de nuevo al usuiario que inserte un nombre valido.

Cada que el udiario inserte en nombre del jugador que hizo el punto el maracador se va vuelve a imprimir con los datos actualizados, asi hasta que tengamos un ganador.
